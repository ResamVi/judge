# Fakultaet

Schreibe ein Programm, welches nach einer Zahl `n` fragt und n! ausgibt
Beispiel: 5! = 5 * 4 * 3 * 2 * 1

Hinweis: 14! ist eine große Zahl und kommt in der Wertung dran

## Ziel

1. `for`-Loops kennengelernt
2. Erkannt was passiert wenn man eine zu große Zahl in ein `int` speichert
