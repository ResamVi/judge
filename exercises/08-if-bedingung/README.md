## Erinnerung

Die logischen Operatoren
```
Gleichheit              ==
Ungleichheit 	        !=
Weniger als 	        <
Gleich oder weniger     <=
Mehr als                >
Gleich oder mehr     	>=
```

Ergebnisse von logischen Operatoren sind Booleans
```
a != 4 // true
a > 5  // false

"apple" < "banana"  // true
"apple" > "banana"  // false
```

If-Bedingungen
```
if zahl > 0 {
    return "Positiv"
} else if zahl < 0 {
    return "Negativ"
} else {
    return "Null"
}
```
