# Judge angreifen

Alle maliziöse Versuche, den Judge zu verlangsamen, zum Absturz zu bringen, Kontrolle zu übernehmen etc. sollten
unter dieser Aufgabe gemacht werden. Gewinner kommen in die Hall of Fame.

## Hall of Fame

1. Julien (14. November 2025)
```
Musste schnell erkennen, dass Programme die nicht aufhören zu laufen, den Judge für die Ewigkeit Arbeiten lassen.
Lässt man 100 von denen Laufen dann war's das.
```

2. Fabian (16. November 2025)
```
Hat beim Beta-Test in einer Endlosschleife Goroutinen erstellt die jede 1GB an Speicher allokieren.

extern heruntergefahren werden.
```

## Ratschläge

Was Hacker machen ist im ersten Schritt sich Zugriff zu einem System zu holen, um im zweiten Schritt
Schadcode auszuführen. 

Jeder Benutzer vom Judge hat quasi den ersten Schritt erreicht und muss jetzt herausfinden wie der 
Schadcode aussehen könnte.
