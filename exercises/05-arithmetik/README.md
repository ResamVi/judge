## Erinnerung

Operationen
```
Operator
+ 	4 + 6 == 10
- 	15 - 10 == 5
* 	2 * 3 == 6
/ 	13 / 3 == 4
% 	13 % 3 == 1
```

Kurzvariante
```
a := 1
a += 2 // gleich wie: a = a + 2
```

Wichtig!
```
var x int = 10
fmt.Println(x / 6) // Ausgabe: 1

var y float32 = 10.0
fmt.Println(y / 6) // Ausgabe: 1.6666666
```
